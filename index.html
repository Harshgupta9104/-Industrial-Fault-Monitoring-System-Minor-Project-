🔍 About the Project

Our system uses multiple sensors integrated with a NodeMCU to monitor environmental conditions in real-time. The data is sent via MQTT and displayed on a live dashboard.

  • 🌡️ Temperature & Humidity via DHT22
  • 🔥 Fire Detection using Flame Sensor
  • ☠️ Harmful Gas Monitoring using MQ135
  • 🏃 Motion Detection using PIR Sensor
  • 🔔 Buzzer & LED for Local Alerts

System Architecture: Sensors → NodeMCU → MQTT → Dashboard
